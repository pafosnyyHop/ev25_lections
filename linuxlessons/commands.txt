linux commands
ctrl + alt + t - терминал

~ -  символ  домашней директории
/ - символ корневой директории 

.. -  обозначают предыдущую директорию
. - обозначают текущую директорию

1. ls(list) - комманда для вывода всех директорий и файлов в текущей директории в терминале

2. cd(change directory) - комманда для изменения одной директории на другую 
        сd <название директории>
        
3. mkdir (make directory) - cоздат указанную директорию в рабчей директории в терминале

4. touch - создает указанный файл

5. rm - удаление файла

6. rmdir - удаление директории

7. pwd (present workink  - выводит путь до текущей директории

8. rm -rf - принудительное удаление 

9. ls -a - выводить список скрытыхттак и обычных файлов идиректорий

10. nano - откроеся файл для редактирования в графическом интерфейсе 
*если нет файла, то nano попытается создать новый файл

11. cat <file_name> - выводит содержимое файла в терминале

12. echo - команда для быстрой записи текстовой информации в какой-либо файл.
    echo текст_для_записи  > название_файла
    
13. mv (move) - команда для переименовывания файла либо директории, а так же для перемещения файлов и директории
переименовывание: mv john jonh
                 mv (старое_название_файла) (новое_название_файла) 
перемещение: mv john ~/Рабочий стол/                 
             mv (file/directory) (directory)/
cp - копирование !
